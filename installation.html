Installation

Foodomaa comes with an easy installation module.

Requirements

Foodomaa requires PHP v7.2 or up, and the following PHP extensions:

  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension
  • Ctype PHP Extension
  • JSON PHP Extension

Good News: Most of the PHP hosting services comes pre-installed with theses extensions, so no need to worry about that. ðŸ™Œ


VERY IMPORTANT

Foodomaa can only be installed on the root of your domain. For Example. www.yourdomain.com OR yourdomain.com
It CANNOT be installed in the subdirectory of your current domain. For Example: www.yourdomain.com/my-new-website

If you need to install Foodomaa locally, please follow the Installation Procedure (on Local System)
We have only tested local installation of foodomaa on Windows 7/8/10 using XAMPP.
Foodomaa will NOT work using php artisan serve


Foodomaa requires an SSL certificate to be installed on your domain to work with all the services. A FREE SSL certificate can be easily installed on your domain using sslforfree.com
Please follow "Install SSL Certificate on Shared Hosting for FREE" video to know more.

Installation Procedure (On Shared Hosting)

Please follow the above video for detailed installation walkthrough for installtion on shared hosting.
Following steps needs to be followed to install the script properly on a shared hosting platform.

  • Step 1. Download the zip file from CodeCanyon.

  • Step 2. Extract the zip file on your system.

  • Step 3. Navigate to "Upload-This" folder.

  • Step 4. Select the file "UPLOAD_THIS.zip" and upload it to the public_html directory of your hosting.

  • Step 5. Extract the contents of the uploaded zip file to the same directory i.e public_html.

  • Step 6. Assuming your website is yourdomain.com, and you have already setup a SSL certificate, navigate to https://yourdomain.com/public/install

Installation Procedure (on Local System)

Please follow the above video for detailed installation walkthrough for installation on localhost on a Windows 7/8/10 OS using XAMPP
Following steps needs to be followed to install the script properly.

  • Step 1. Download the zip file from CodeCanyon.

  • Step 2. Extract the zip file on your system.

  • Step 3. Make the edits in the apache configuration file as shown in the above video.

  • Step 4. Navigate to xampp > htdocs and create a new folder named foodomaa

  • Step 5. Select the downloaded zip file "UPLOAD_THIS.zip" and extract it into the newly create folder i.e "foodomaa"

  • Step 6. Create a database using phpMyAdmin.

  • Step 7. Open up a browser and navigate to http://localhost/public/install

Database

Follow the following steps to create a database:

  • Step 1. Login to your cPanel provided by your hosting provider.

  • Step 2. Go to MYSQL Database Wizard

  • Step 3. Enter a database name.

  • Step 4. Create a username and password for the database user.

  • Step 5. Click All Privileges in the permission section

  • Step 6. Go back to the Foodomaa Setup Wizard and enter the database credentials which you just created.

Troubleshooting

The front store is built with ReactJS and a Progressive Web App is implemented using a Service Worker.

The Service Worker caches all the static files like images, javascripts, css, website settings and even html.
So, if your website doesn't quite look like the one on the demo or the walkthrough video, you probably have some cache issues.
Luckily, there is a very easy solution for this.

  • Step 1. Right click on anywhere on the page and select "Inspect Element".
  • Step 2. Navigate to Application > Clear Storage and then click on "Clear site data"

Doing this will hard reset the cache and reset the ServiceWorker to fetch new files and settings.

Fix 404 Errors

If you are getting a 404 error for URLs:

  1. Download the htaccess zip file.
  2. Extract the zip on the root directory of your website.
Note: Already a htaccess file is provide, the original htacces file is to be replaced with this new file. Download htaccess Zip File
